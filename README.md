## node version

Please has installed a node version of 18.19.0 or higher

## install dependencies

```sh
npm install
```

### run app

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### run unit rests

```sh
npm run test:unit
```

### run lint

```sh
npm run lint
```
